
# Retail Sales EDA

## Project Overview
This project performs exploratory data analysis on a retail sales dataset to
understand sales trends, top products, and revenue patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Questions Answered
- Which product category generates the highest revenue?
- How do monthly sales change over time?
- What is the average order value?

## Dataset
Retail sales transaction data containing:
- Date
- Product Category
- Quantity
- Price
- Total Sales

## Key Insights
- Electronics contributed the highest revenue.
- Sales peak during festive months.
- Average order value increased by 12% over time.


## Learning Outcome
Improved understanding of data cleaning, visualization, and business-focused analysis.
